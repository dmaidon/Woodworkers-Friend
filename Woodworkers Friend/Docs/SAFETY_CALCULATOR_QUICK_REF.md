### Risk Assessment Matrix

|-------------|----------------|------------|--------------|
| < 3" | Any | **CRITICAL** | 2 push sticks, push block, featherboards MANDATORY |

| 6"-12" | 0.75"+ | **MODERATE** | Push stick for last 12", guard recommended |
| 12"+ | 0.75"+ | **LOW** | Push stick for final pass, standard precautions |

### The 6-Inch Rule
**NEVER let your hands come within 6 inches of the blade while it's running.**

### Critical Safety Zones
- **Red Zone (0-3")** - Blade to stock - NEVER put hands here
- **Yellow Zone (3-6")** - High risk - Use push devices only
- **Green Zone (6"+)** - Safe hand placement

## Emergency Procedures

### If Kickback Occurs
1. **Let go of the workpiece** - Don't fight it
2. **Step aside** - Don't be in line with the projectile
3. **Turn off saw immediately**
4. **Inspect workpiece and blade** for damage
5. **Identify cause** before continuing

### If Injury Occurs
1. **Turn off tool immediately** - Emergency stop
2. **Apply direct pressure** to bleeding
3. **Call for help / 911** for serious injuries
4. **Keep first aid kit accessible** in shop
5. **Don't work alone** if possible

### If Blade Binds
1. **Turn off saw** - Don't try to force it
2. **Wait for blade to stop completely**
3. **Unplug the saw**
4. **Carefully remove workpiece**
5. **Check alignment and setup** before restarting

## Key Takeaways

### Router Safety
- **Large bits = Low RPM** - Never exceed manufacturer specifications
- **Variable speed router essential** for bits > 2" diameter
- **Inspect bits before use** - Cracks can cause catastrophic failure

### Saw Safety
- **Blade height matters** - Follow operation-specific recommendations
- **Push sticks save fingers** - Use them religiously
- **Guards are there for a reason** - Keep them in place

### Dust Collection
- **Your lungs can't filter wood dust** - They're not designed for it
- **CFM is crucial** - Undersized collectors are worthless
- **Fine dust is invisible** - And most dangerous
