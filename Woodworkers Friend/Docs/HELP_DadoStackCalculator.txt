#TITLE:Dado Stack Calculator

##SECTION:Overview|Calculate optimal blade combinations for dado stack setups

The Dado Stack Calculator helps you determine the perfect combination of dado blades and chippers to achieve your desired width cut. Not all dado sets come with comprehensive instructions, so this calculator uses a smart algorithm to find the best blade configuration.

###SUBTITLE:Why Use This Calculator?

*BULLET:Eliminates trial and error when setting up dado stacks
*BULLET:Finds optimal blade combinations automatically
*BULLET:Shows alternative solutions if available
*BULLET:Prevents over-cutting by calculating exact width
*BULLET:Supports both imperial and metric units
*BULLET:Save and load custom dado set configurations

##SECTION:Input Fields|What you need to enter

###SUBTITLE:Desired Width
Enter the target width you want to cut. This should match the thickness of the material you're housing (for shelf dados) or the desired groove width.

*BULLET:Range: 0.125" to 2.0" (3mm to 50mm)
*BULLET:Increment: 0.0625" (1/16" or 1.6mm)
*BULLET:Common values: 1/2", 5/8", 3/4" for plywood shelves

###SUBTITLE:Units
Select your preferred measurement system:
*BULLET:Inches (Imperial) - Standard US woodworking
*BULLET:Millimeters (Metric) - European standard

###SUBTITLE:Kerf Width
The thickness of each blade's cutting edge. Most dado blades have a kerf of 1/8" (0.125"), but check your specific set.

*BULLET:Default: 0.125" (1/8")
*BULLET:Range: 0.001" to 0.25"
*BULLET:Affects stacking calculations and precision

###SUBTITLE:My Dado Set
Check the blades and chippers available in your dado set:

*BULLET:Two 1/8" Outer Blades - Required (always checked)
*BULLET:1/16" Chipper - Common in premium sets
*BULLET:1/8" Chipper - Standard size
*BULLET:3/16" Chipper - Less common
*BULLET:1/4" Chipper - Large chipper
*BULLET:Shims (0.004", 0.008") - For micro-adjustments

!WARNING:The outer blades must always be included - they're required for safe dado stack operation

##SECTION:Tips and Best Practices|Getting the best results

###SUBTITLE:Accuracy Tips

*BULLET:Measure your actual blade widths - they may differ from nominal
*BULLET:Clean blades before stacking - sawdust affects width
*BULLET:Use shims for final micro-adjustments
*BULLET:Test cut on scrap before final workpiece
*BULLET:Stack blades in the order shown by calculator

###SUBTITLE:Safety Reminders

!WARNING:Never exceed maximum blade height for your saw
!WARNING:Always use outer blades - required for balance and safety
!WARNING:Check that arbor length accommodates your stack
!WARNING:Tighten arbor nut properly - loose stacks are dangerous
!WARNING:Make test cuts to verify width before cutting workpiece

###SUBTITLE:Common Mistakes to Avoid

&PROBLEM:Stack is too wide for arbor
&SOLUTION:Remove smallest chipper or use shims instead of full chippers

&PROBLEM:Cut width doesn't match calculation
&SOLUTION:Verify blade widths match catalog specs - some vary by manufacturer

&PROBLEM:Rough cut bottom
&SOLUTION:Ensure outer blades are positioned correctly - flat sides out

&PROBLEM:Excessive vibration
&SOLUTION:Check blade alignment and arbor washer placement

##SECTION:Tolerance and Precision|Understanding the numbers

The calculator uses a default tolerance of ±0.005" (5 thousandths of an inch) when finding optimal combinations.

=FORMULA:Error = Total Width - Target Width

###SUBTITLE:Acceptable Tolerances

*BULLET:Exact match: ±0.001" or less (1 thousandth)
*BULLET:Excellent: ±0.002" to ±0.005" (2-5 thousandths)
*BULLET:Acceptable: ±0.006" to ±0.010" (6-10 thousandths)
*BULLET:Too loose: > ±0.010" (may require shims)

?NOTE:For plywood dado joints, aim for snug fit - too tight and plywood may split, too loose and joint is weak

##SECTION:Troubleshooting|Common issues and solutions

&PROBLEM:No solution found
&SOLUTION:Check that enough blades are selected. Try different target width or add custom sizes.

&PROBLEM:Calculator shows multiple alternatives with same width
&SOLUTION:This is normal - algorithm found different blade orders yielding same result. Choose simplest stack.

&PROBLEM:Error is always positive (too wide)
&SOLUTION:Your target may be too narrow for available blades. Add smaller chippers or use shims.

&PROBLEM:Can't add custom blade
&SOLUTION:Check format - must be "Description (0.XXX")" with width in decimal inches.

&PROBLEM:Results don't match actual cut
&SOLUTION:Verify kerf width setting matches your blades. Clean blades and check for debris between chippers.

###SUBTITLE:Algorithm Performance

*BULLET:Pruning: Early termination on exact matches
*BULLET:Memory: Stores top 3 solutions only (minimal overhead)
*BULLET:Tolerance: Configurable (default ±0.005")
