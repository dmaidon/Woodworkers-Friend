# Version Information

## Current Version
**Woodworker's Friend v26.1.28.344**

Released: January 2026

## What's New in This Version

### Help System Enhancement
- Added embedded markdown-based help system
- Help content now loads from embedded resources
- Improved help navigation and formatting
- Better error messages when resources not found

### Features
- Comprehensive calculator suite for woodworking projects
- Support for multiple drawer construction methods
- Door calculator with inset and overlay options
- Board feet and material cost calculations
- Epoxy pour volume calculator
- Wood movement calculations
- Unit conversion tools

### User Interface
- Dark and light theme support
- Export to CSV, Text, and HTML formats
- Preset management system
- Input validation with helpful feedback
- Keyboard shortcuts for efficient workflow

## System Requirements

### Minimum Requirements
- **Operating System**: Windows 10 or later
- **.NET**: .NET 10.0 or later
- **Display**: 1280x720 minimum resolution
- **Memory**: 512 MB RAM
- **Storage**: 50 MB available space

### Recommended
- **Operating System**: Windows 11
- **Display**: 1920x1080 resolution
- **Memory**: 2 GB RAM

## About the Application

Woodworker's Friend was developed to help woodworkers of all skill levels plan and execute their projects with precision. The application combines traditional woodworking knowledge with modern calculation tools to make project planning easier and more accurate.
