# Themes

## Overview
Woodworker's Friend supports both Light and Dark themes to match your working environment and personal preference.

## Available Themes

### Light Theme
**Best for:** Bright environments, traditional appearance

**Features:**
- White background
- Dark text for contrast
- Standard Windows control styling
- Clear visibility in daylight
- Professional appearance

**When to Use:**
- Well-lit workshops
- Daytime use
- Bright screen settings
- Traditional interface preference

### Dark Theme
**Best for:** Low-light environments, reduced eye strain

**Features:**
- Dark background
- Light text for readability
- Reduced screen glare
- Comfortable for extended use
- Modern appearance

**When to Use:**
- Evening work sessions
- Dimly lit spaces
- Reducing eye fatigue
- Modern interface preference

## Switching Themes

### Method 1: Keyboard Shortcut
- Press **Ctrl+T** to toggle instantly
- Fastest way to switch
- Works from any tab
- No menu navigation needed

### Method 2: View Menu
1. Click **View** menu
2. Select **Toggle Theme**
3. Theme switches immediately

### Method 3: Settings
- Access application settings
- Choose preferred theme
- Set as default for startup

## Theme Features

### Consistent Design
Both themes maintain:
- Same layout and organization
- Identical functionality
- Similar color coding for validation
- Consistent button placement

### Color Coding
**Light Theme:**
- Errors: Red highlights
- Success: Green highlights
- Warnings: Yellow highlights
- Information: Blue highlights

**Dark Theme:**
- Errors: Red highlights (adjusted for dark bg)
- Success: Green highlights (adjusted for dark bg)
- Warnings: Yellow highlights (adjusted for dark bg)
- Information: Blue highlights (adjusted for dark bg)

### Readability
Both themes optimized for:
- Clear text contrast
- Readable font sizes
- Distinct control borders
- Visible focus indicators

## Troubleshooting

### Theme Won't Switch
- Try keyboard shortcut (Ctrl+T)
- Check View menu
- Restart application
- Reset preferences if needed

### Colors Look Wrong
- Verify Windows theme settings
- Check display color profile
- Update graphics drivers
- Ensure latest app version

### Preference Not Saved
- Check write permissions
- Verify settings folder
- Restart application
- Re-select and test
