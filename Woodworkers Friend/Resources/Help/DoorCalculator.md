# Door Calculator

## Overview
The Door Calculator helps you design custom cabinet doors for both face frame and frameless cabinets.

## Door Styles

### Inset Doors
Doors that fit inside the cabinet opening, flush with the face frame.

**Required Inputs:**
- **Cabinet Opening Width**: Inside dimension of cabinet
- **Cabinet Opening Height**: Vertical opening size
- **Reveal**: Gap between door and frame (typically 1/16")
- **Material Thickness**: Door panel thickness

### Overlay Doors
Doors that overlap the cabinet face.

**Required Inputs:**
- **Cabinet Opening Width**: Inside dimension
- **Cabinet Opening Height**: Vertical opening size
- **Overlay Amount**: How much door overlaps (typically 1/2")
- **Material Thickness**: Door panel thickness

## Results Provided
- Door width and height
- Rail and stile dimensions
- Panel dimensions for raised panel doors
- Hinge placement recommendations

## Tips
- Allow 1/16" clearance minimum for smooth operation
- Full overlay typically uses 1/2" to 5/8" overlay
- European hinges require specific boring dimensions
- Consider grain direction on frame components
