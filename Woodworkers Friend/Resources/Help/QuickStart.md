# Quick Start Guide

## First Launch

When you first start Woodworker's Friend:

1. **Choose Your Theme**
   - Light theme for bright environments
   - Dark theme for reduced eye strain
   - Switch anytime with Ctrl+T

2. **Explore the Tabs**
   - Each tab is a different calculator
   - Start with one that matches your current project
   - No need to configure anything to get started

3. **Try a Calculation**
   - Enter some sample values
   - Click Calculate
   - See immediate results

## Your First Calculation

Let's build a simple drawer:

### Step 1: Open Drawer Calculator
- Click the **Drawers** tab at top

### Step 2: Enter Dimensions
- **Cabinet Width**: 18 (inches)
- **Cabinet Height**: 4 (inches)
- **Cabinet Depth**: 20 (inches)
- **Material Thickness**: 0.5 (inches)

### Step 3: Choose Method
- Select "Face Frame Standard" from dropdown

### Step 4: Calculate
- Click the **Calculate** button
- Results appear immediately

### Step 5: Review Results
- Front dimensions
- Side dimensions
- Back dimensions
- Bottom panel size

### Step 6: Export (Optional)
- Click **Export** button
- Choose CSV, TXT, or HTML
- Save for reference

## Essential Tips

### Input Tips
- Use Tab key to move between fields
- Press Enter to calculate
- Values must be positive numbers
- Use decimal point (not comma)

### Validation
- Red highlights mean invalid input
- Hover for error explanation
- Fix errors before calculating
- Status bar shows summary

### Saving Work
- Export results to file
- Save as preset for reuse
- Include in project documentation
- Keep organized by project

## Common Questions

### Q: Do I need to save my settings?
A: Settings auto-save when you exit. Calculators remember last values.

### Q: Can I use metric measurements?
A: Use Unit Conversions tab to convert between imperial and metric.

### Q: How accurate are the calculations?
A: Results are mathematically precise. Always add tolerance for real-world conditions.

### Q: Can I print the results?
A: Export to HTML for formatted printing, or use Print function.

### Q: What if I make a mistake?
A: Simply change the input value and recalculate. Previous results update instantly.
