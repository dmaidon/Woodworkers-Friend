# Understanding the Interface

## Main Window Layout

The Woodworker's Friend application uses a tabbed interface for easy navigation between different calculators and tools.

## Tab Organization

### Calculator Tabs
- **Drawers** - Drawer box dimension calculator
- **Doors** - Cabinet door calculator
- **Board Feet** - Lumber volume and cost calculator
- **Epoxy** - Epoxy pour volume calculator
- **Polygon** - Multi-sided figure calculator
- **Wood Movement** - Seasonal expansion/contraction calculator
- **Unit Conversions** - Metric/Imperial converter

### Utility Tabs
- **Help** - This comprehensive help system
- **Settings** - Application preferences
- **About** - Version and credits information

## Input Areas

### Common Controls
- **TextBox Fields**: Enter numeric values
- **ComboBox Dropdowns**: Select from predefined options
- **Radio Buttons**: Choose one option from a group
- **Checkboxes**: Enable/disable options

### Input Validation
- **Red highlights**: Invalid or out-of-range values
- **Tooltips**: Hover for field descriptions
- **Error messages**: Specific guidance on what's wrong

## Results Display

### Output Sections
- **Results Area**: Calculated values displayed clearly
- **Units**: Always shown with values
- **Precision**: Appropriate decimal places for woodworking
- **Labels**: Descriptive names for each result

### Visual Feedback
- **Colors**: Green for success, red for errors
- **Formatting**: Bold for important values
- **Grouping**: Related results organized together

## Buttons and Controls

### Common Buttons
- **Calculate**: Performs the calculation
- **Clear**: Resets all inputs
- **Export**: Saves results to file
- **Load Preset**: Loads saved values
- **Save Preset**: Saves current values

### Navigation
- **Tab Bar**: Click to switch calculators
- **Keyboard**: Use Ctrl+Tab to cycle tabs
- **F1**: Context-sensitive help

## Status Bar

Located at bottom of window:
- **Current operation**: What's happening
- **Validation status**: Input errors summary
- **Theme indicator**: Light/Dark mode
- **Version**: Application version number

### Light Theme
- White background
- Dark text

### Dark Theme

**Switch themes:** Ctrl+T or View > Toggle Theme

## Tips for Navigation

### Efficient Workflow
1. Select calculator tab
2. Enter values (use Tab to move between fields)
3. Click Calculate or press Enter
4. Review results
5. Export if needed

### Keyboard Shortcuts
- Use Tab and Shift+Tab to navigate fields
- Press Enter to calculate
- Ctrl+E to export
- F1 for help on current calculator

### Mouse Usage
- Click directly in fields to enter values
- Use dropdown arrows for combo boxes
- Right-click for context menus (where available)
- Drag window edges to resize
