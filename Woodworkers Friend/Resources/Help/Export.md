# Exporting Results

## Overview
Export your calculations to save, share, or print your project data.

## Supported Formats

### CSV (Comma-Separated Values)
**Best for:** Spreadsheet applications like Excel

**Features:**
- Opens directly in Excel, Google Sheets, LibreOffice
- Easy to manipulate and analyze data
- Compatible with most database programs
- Lightweight file format

**Use when:** You need to perform additional calculations or create charts

### Text (TXT)
**Best for:** Simple documentation and printing

**Features:**
- Human-readable plain text
- Opens in any text editor
- Smallest file size
- Easy to email or share

**Use when:** You need a quick reference or printout

### HTML
**Best for:** Web viewing and formatted reports

**Features:**
- Opens in any web browser
- Preserves formatting and colors
- Can include project notes
- Professional appearance

**Use when:** Creating project documentation or portfolios

## How to Export

1. **Complete Your Calculations**
   - Enter all required inputs
   - Click Calculate to generate results
   - Verify results are correct

2. **Click Export Button**
   - Located on each calculator tab
   - Or use Ctrl+E keyboard shortcut

3. **Choose Format**
   - Select CSV, TXT, or HTML
   - Each has different advantages

4. **Select Location**
   - Choose where to save the file
   - Use descriptive filename
   - Consider organizing by project

5. **Verify Export**
   - Open file to confirm contents
   - Check all values exported correctly

## What Gets Exported

### Included Data
- All input values entered
- Calculated results
- Selected method or options
- Timestamp of calculation
- Calculator type and version

### Not Included
- Visual drawings or diagrams
- Application settings
- Preset information
- Help content

## Tips and Best Practices

### File Naming
Use descriptive names that include:
- Project name
- Calculator type
- Date
- Version if iterating

**Example:** `KitchenCabinet_Drawer_2026-01-29_v2.csv`

### Organization
- Create project folders
- Keep all related calculations together
- Export iterations to track changes
- Include export files with project photos

### When to Export
- Before starting construction
- After making design changes
- When sharing with others
- For project documentation
- As backup of calculations

### Batch Operations
- Export multiple calculators for same project
- Keep exports together in project folder
- Use consistent naming scheme
- Date stamp for version control

## Troubleshooting

### Export Fails
- Check write permissions on destination folder
- Ensure sufficient disk space
- Close any open files with same name
- Try different location or format

### Missing Data
- Verify all calculations completed before export
- Check that required fields were filled
- Recalculate and export again

### Cannot Open File
- Verify correct application for file type
- Check file extension matches content
- Try different export format
- Ensure export completed successfully
