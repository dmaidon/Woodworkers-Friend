# Getting Started with Woodworker's Friend

Welcome to **Woodworker's Friend** - your comprehensive woodworking calculator and planning tool!

## What Can You Do?

This application helps you calculate dimensions, materials, and perform conversions for various woodworking projects:

### Calculators
- **Drawer Calculator** - Calculate drawer heights using mathematical progressions
- **Door Calculator** - Design cabinet doors with precise rail and stile dimensions
- **Board Feet Calculator** - Calculate lumber requirements and costs
- **Epoxy Pour Calculator** - Determine epoxy volumes for river tables
- **Polygon Calculator** - Calculate angles and dimensions for polygon projects
- **Table Tip Calculator** - Determine tipping force for furniture stability

### Joinery Tools
- **Mortise & Tenon** - Calculate proper joint dimensions
- **Dovetails** - Design dovetail joints with correct angles
- **Box Joints** - Calculate finger joint spacing
- **Dado Joints** - Determine dado depth and width

### Wood Movement & Structural
- **Wood Movement Calculator** - Predict expansion/contraction and plan gaps
- **Shelf Sag Calculator** - Design safe shelves with proper load capacity

### Optimization
- **Cut List Optimizer** - Minimize material waste with intelligent cutting patterns

### Conversions

## Common Workflows

### Building Cabinet Doors
1. Open **Doors** tab
2. Enter opening dimensions
3. Select door style (overlay/inset, 1 or 2 doors)
4. Review rail/stile/panel dimensions
5. Export results

### Designing Shelves
1. Open **Wood Movement** tab ‚Üí **Shelf Sag** sub-tab
2. Enter span, load, material, dimensions
3. Add edge stiffeners if needed
4. Check safety factor and load limits
5. Adjust until green (safe)

### Planning Drawer Stack
1. Open **Drawers** tab
2. Enter total height and drawer count
3. Choose progression type (uniform, golden ratio, etc.)
4. Review individual drawer heights
5. Print or export results
