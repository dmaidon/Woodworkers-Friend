# Getting Started with Woodworker's Friend

## Welcome!
Woodworker's Friend is your comprehensive woodworking calculator and planning tool. This application helps you calculate dimensions, materials, and perform conversions for various woodworking projects.

## What Can You Do?

### Calculators
- **Drawer Calculator**: Calculate drawer dimensions for 10 different construction methods
- **Door Calculator**: Design cabinet doors (inset and overlay styles)
- **Board Feet Calculator**: Calculate lumber volume and costs
- **Epoxy Pour Calculator**: Determine epoxy quantities for river tables and pours
- **Polygon Calculator**: Calculate angles and dimensions for polygonal projects

### Conversions
- **Unit Conversions**: Convert between imperial and metric measurements
- **Fraction to Decimal**: Quick fraction conversions
- **Table Tipping Force**: Calculate stability for tables and furniture

### Features
- Export results to CSV, Text, or HTML formats
- Save and load presets for common projects
- Input validation with helpful feedback
- Dark and light theme support

## Quick Start
1. Select a calculator from the tab bar
2. Enter your measurements
3. Click Calculate to see results
4. Use Export to save your calculations
5. Access Help anytime for detailed instructions

## Tips
- Hover over input fields for tooltips
- Use Tab key to move between fields quickly
- Save frequently used settings as presets
- Check validation messages if inputs are highlighted

Explore each calculator's help section for detailed instructions!
