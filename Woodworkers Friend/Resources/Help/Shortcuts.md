# Keyboard Shortcuts

## General Navigation

- **Tab** - Move to next input field
- **Shift+Tab** - Move to previous input field
- **Enter** - Activate focused button or calculate
- **Escape** - Clear current form or close dialog
- **F1** - Open context-sensitive help

## Calculator Operations

- **Ctrl+C** - Copy selected results
- **Ctrl+V** - Paste into input field
- **Ctrl+A** - Select all in focused field
- **Ctrl+Z** - Undo last input change

## File Operations

- **Ctrl+S** - Save current preset
- **Ctrl+O** - Open/Load preset
- **Ctrl+E** - Export results
- **Ctrl+P** - Print results

## View Controls

- **Ctrl+T** - Toggle between Light and Dark themes
- **Ctrl++** - Increase text size (where supported)
- **Ctrl+-** - Decrease text size (where supported)
- **F11** - Toggle full screen mode

## Application

- **Ctrl+H** - Open Help tab
- **Ctrl+Q** - Quit application
- **F12** - Open About/Version information

## Calculator-Specific

### Drawer Calculator
- **Ctrl+D** - Draw preview (if available)
- **Ctrl+M** - Cycle through methods

### Quick Entry

- **Arrow Keys** - Navigate between fields in grid layouts
- **Page Up/Down** - Increment/decrement numeric values
- **Home/End** - Jump to first/last field in section

## Tips

- Hold **Ctrl** while clicking buttons for additional options
- Use **Tab** for efficient data entry without mouse
- Most fields accept keyboard-pasted values
- **F1** opens help relevant to current calculator

## Customizing Shortcuts

Some shortcuts can be customized in Settings (planned feature).
