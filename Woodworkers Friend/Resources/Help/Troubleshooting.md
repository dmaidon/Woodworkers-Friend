# Troubleshooting

## Common Issues and Solutions

### Calculator Not Updating
**Problem:** Results don't update when you change inputs.
**Solution:**
- Click the Calculate button after changing values
- Check that all required fields are filled
- Verify input values are within valid ranges

### Invalid Input Errors
**Problem:** Input fields show validation errors.
**Solution:**
- Check for negative numbers where positive required
- Verify decimal format (use period, not comma)
- Ensure values are within acceptable ranges
- Remove any non-numeric characters

### Export Fails
**Problem:** Cannot export results to file.
**Solution:**
- Ensure you have write permissions to the folder
- Close any open files with the same name
- Check available disk space
- Try a different file format (CSV, TXT, HTML)

### Preset Not Loading
**Problem:** Saved preset doesn't load correctly.
**Solution:**
- Verify preset file exists in Settings folder
- Check file is not corrupted
- Ensure preset was created with compatible version
- Try recreating the preset

### Display Issues
**Problem:** Text is too small or colors are wrong.
**Solution:**
- Try switching between Light and Dark themes
- Adjust Windows display scaling settings
- Update graphics drivers
- Restart the application

### Calculation Seems Wrong
**Problem:** Results don't match expectations.
**Solution:**
- Double-check input values and units
- Verify the correct calculation method is selected
- Review the formulas in calculator-specific help
- Consider material shrinkage and tolerances

## Getting Additional Help

If problems persist:
1. Check the Recent Updates section for known issues
2. Review help for the specific calculator you're using
3. Verify your inputs against examples in help
4. Contact support with specific error details

## Error Logs

Application logs are stored in:
- Windows: `%APPDATA%\WoodworkersFriend\Logs`

Include log files when reporting issues for faster resolution.
