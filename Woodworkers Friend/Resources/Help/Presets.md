# Using Presets

## Overview
Presets allow you to save commonly used calculator settings for quick recall later.

## What Are Presets?

Presets store all input values for a specific calculator, allowing you to:
- Save frequently used dimensions
- Maintain standard project sizes
- Share settings with others
- Quickly switch between configurations

## Creating a Preset

1. **Enter Your Values**
   - Fill in all calculator inputs
   - Verify values are correct
   - Calculate to ensure they work

2. **Save the Preset**
   - Click **Save Preset** button
   - Or press Ctrl+S
   - Give it a descriptive name
   - Add optional notes

3. **Confirm Save**
   - Preset stored in Settings folder
   - Available for future use
   - Can be edited or deleted

## Loading a Preset

1. **Open Preset Dialog**
   - Click **Load Preset** button
   - Or press Ctrl+O
   - Browse available presets

2. **Select Preset**
   - Click desired preset
   - Preview values shown
   - Click Load to apply

3. **Values Applied**
   - All inputs populated
   - Ready to calculate
   - Modify as needed

## Managing Presets

### Editing Presets
- Load the preset
- Modify values as needed
- Save with same or new name

### Deleting Presets
- Open preset manager
- Select preset to remove
- Confirm deletion

### Organizing Presets
- Use descriptive names
- Include dimensions in name
- Group by project type
- Date stamp versions

## Preset Naming Tips

### Good Names
- "Kitchen_Upper_12x30"
- "Drawer_Standard_18x6x20"
- "Door_Inset_14x28"
- "BoardFeet_1x6_Pine"

### Include
- Project type
- Key dimensions
- Material if relevant
- Version number if iterating

## Where Presets Are Stored

**Location:** Settings folder in application directory

**Format:** Configuration files (.preset)

**Backup:** Copy entire Settings folder to backup presets

## Sharing Presets

### Export
- Presets stored as files
- Can be copied to USB drive
- Email to collaborators
- Share in cloud storage

### Import
- Copy preset files to Settings folder
- Restart application
- Presets appear in list
- Ready to use

## Common Preset Uses

### Standard Drawer Sizes
Save presets for:
- 12", 15", 18", 21", 24" wide drawers
- Various heights (4", 6", 8", 10")
- Different construction methods

### Cabinet Doors
Presets for:
- Standard door sizes
- Overlay amounts
- Reveal distances
- Frame dimensions

### Lumber Calculations
Save for:
- Common board dimensions
- Favorite wood species
- Standard thicknesses
- Cost tracking

## Tips and Best Practices

### Organization
- Create project-specific presets
- Use consistent naming
- Delete unused presets
- Keep library manageable

### Updating
- Review presets periodically
- Update costs if saved
- Revise based on experience
- Document changes

### Documentation
- Add notes to presets
- Include special considerations
- Note any adjustments needed
- Reference project photos

## Troubleshooting

### Preset Won't Load
- Check file not corrupted
- Verify compatible version
- Look for missing dependencies
- Try recreating preset

### Preset List Empty
- Check Settings folder exists
- Verify file permissions
- Look for .preset files
- Restart application

### Values Don't Match
- Confirm correct preset loaded
- Check for calculator updates
- Verify units consistent
- Recalculate after loading
