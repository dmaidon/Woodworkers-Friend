# Fraction to Decimal Converter

## Overview
Convert fractional inches to decimal format or vice versa.

## Common Conversions

### Sixteenths
- 1/16" = 0.0625"
- 1/8" = 0.125"
- 3/16" = 0.1875"
- 1/4" = 0.25"
- 5/16" = 0.3125"
- 3/8" = 0.375"
- 7/16" = 0.4375"
- 1/2" = 0.5"

### Thirty-Seconds
- 1/32" = 0.03125"
- 3/32" = 0.09375"
- 5/32" = 0.15625"
- 7/32" = 0.21875"

## How to Use

### Fraction to Decimal
1. Enter numerator (top number)
2. Enter denominator (bottom number)
3. See decimal result

### Decimal to Fraction
1. Enter decimal value
2. Choose precision (1/16, 1/32, 1/64)
3. See closest fraction
